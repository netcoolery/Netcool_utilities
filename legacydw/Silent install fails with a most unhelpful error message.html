Silent install fails with a most unhelpful error message
Published: July 5th, 2013

The customer was doing a silent installation of OMNIbus 7.4, but the installer would exit with this message:

The installer cannot run on your configuration. It will now quit

Good grief!  Could this be any less informative?   Fortunately  google came to the rescue and served this up (when presented with the entire error message.)

http://www-01.ibm.com/support/docview.wss?uid=swg21430495

The tech note in question is for a different installation, but the symptom was the same and indeed it was the same mistake.  The customer had

  LICENSE_ACCEPTED=trus

in the silent install response file.
