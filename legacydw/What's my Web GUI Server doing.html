What's my Web GUI Server doing?
Published: May 1st, 2013 | Last Updated: May 24th, 2013

HT to Matt Lush for this:

Not sure what version they are on, but as of Web GUI 7.3.1 there is quite a lot of profiling data written to ncw.*.log files - when monitoring the Web GUI server I periodically record the following values from the log as user load is ramped up -

     "Total time spent on updating the ObjectServer"
     "Total time spent on Event Summary Data"
     "Total time spent on Event Data"
     "Total time in the report"

where Event Summary Data is the refreshing dashboards usage and Event Data is the refreshing AELs usage. Usage for each individual user is also written to the log as well I think.
